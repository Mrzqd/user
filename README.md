# Usage

设置以下回购秘密，然后手动触发 “注册应用” 工作流。有关更多详细信息，请阅读 <https:// sl.al/ABzD>。

| Name   | Value                                                   |
| ------ | ------------------------------------------------------- |
| PAT    | Github personal access token with `workflow` permission |
| USER   | E5 admin emails line seperated                          |
| PASSWD | E5 admin passwords line seperated                       |
